== Description ==

Gateway for Leyka donation management system which adds option for donating using bank order or custom  requisites.

Leyka plugin been translated into the following languages:

1. Russian
2. English

== Installation ==

1. Install and activate the plugin.
2. Go to Donates -> Settings and configure settings for gateway.
3. If no donate page exists create new Donate from Donates -> Add New page.
4. Activate the gateway on Donates -> Settings page.
5. For detailed setup instructions, visit the official [Documentation](http://leyka.te-st.ru) page.
